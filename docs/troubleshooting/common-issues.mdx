## Common Issues with SideStore
### AFC was unable to...
#### Cause
Either connection issue or invalid pairing file
#### Resolution
1. Try again
2. Recreate a new pairing file using `jitterbugpair` and load it via the settings menu
### No WiFi or VPN!
#### Cause
Any connection issue with minimuxer causes this issue
#### Resolution
1. Try again (spamming the button usually helps)
2. Restart SideStore
3. Restart WireGuard
4. Start SideStore before WireGuard
5. Recreate a new pairing file using `jitterbugpair` and load it via the settings menu
